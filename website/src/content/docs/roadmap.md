## v0.1 (current)

ark v0.1 ships the complete scene system, extension protocol, ACP integration, supervisor lifecycle, and zellij mux. See [Install](/learn/install/) to get started.

## v0.2 (planned)

- **Extension marketplace** — a public index of community extensions, discoverable via `ark ext search`
- **Additional built-in extensions** — expanded set of shipped extensions beyond status + picker
- **Scene templates** — `ark scene init` with template selection for common workflows

## v0.3 (future)

- **Multi-agent orchestration** — coordinate multiple agents across a single session
- **Custom mux backends** — support for terminal multiplexers beyond zellij
- **Remote sessions** — spawn agents on remote hosts with local UI
